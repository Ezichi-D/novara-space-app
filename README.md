Planet X

Planet Explorer is a React-based web application that allows users to explore planetary data fetched from an external API. The application displays planet names, images, and distances from the sun in a clean and responsive interface.

Live Demo

[View Live Project](https://planet-x-alpha.vercel.app/)

---

Features

* Fetches planetary data from an external API
* Displays planet names, images, and distances from the sun
* Responsive design for desktop and mobile devices
* Smooth scrolling navigation
* Contact form with validation
* Autoplay, muted, looping background video
